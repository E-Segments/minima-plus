Welcome to your new blog! This is a sample post to demonstrate the blog features of this Jekyll theme.

## What's Included

This theme comes with everything you need to build a modern website:

- **Landing pages** with hero sections and feature grids
- **Blog** with categories, tags, and reading time
- **Documentation** with sidebar navigation
- **Dark mode** that respects user preferences
- **Responsive design** that works on all devices

## Customization

You can customize nearly everything through the `_config.yml` file:

```yaml
theme_config:
  colors:
    primary: "indigo"
    secondary: "slate"
  fonts:
    heading: "Inter"
    body: "Inter"
```

## Components

The theme includes reusable components that you can use anywhere:

### Cards

Use cards to highlight features or link to other content.

### Alerts

Draw attention to important information with alerts.

### Buttons

Customizable buttons with different styles and sizes.
